## SECTION 2: COMPETITOR-BY-COMPETITOR BREAKDOWN

### 2.1 Action Network — The Most Comparable Exit

**Overview:** The largest US consumer betting media + data platform. Acquired by Better Collective (Danish media company) for **$240M in 2021**.

| Dimension | Action Network | SORS MAXIMA |
|-----------|---------------|-------------|
| Business model | SaaS ($8–$47/mo) + ad revenue | SaaS only ($49–$199/mo, no ads) |
| Core product | Editorial picks + odds tracker | Simulation engine + AI picks |
| Pick methodology | Human analysts + algo layer | 51-Factor Monte Carlo + AI |
| Simulation depth | None | 1M+ sims per game |
| AI companion | None | GPT-4o grounded in live data |
| Admin intelligence suite | Not user-accessible | Fully built + deployed |
| Price at acquisition | ~8x ARR | Comparable at $1M+ ARR |
| **Exit value** | **$240M** | **Projected $12M–$118M (base-bull)** |

**Key Lesson from Action Network:** Better Collective paid ~$240M for an audience, not the technology. SORS MAXIMA's technology is demonstrably more sophisticated — the opportunity is to acquire the audience and let the technology accelerate that value.

### 2.2 Unabated — Sharp Tools, No Consumer UX

**Overview:** Advanced sharp-bettor analytics platform. Focused on professional/semi-pro bettors. Does not have a simulation engine.

| Dimension | Unabated | SORS MAXIMA |
|-----------|----------|-------------|
| Price | $49–$199/mo | $49–$199/mo |
| Target user | Professional bettors | Serious recreational + serious |
| Simulation engine | None | 51-factor Monte Carlo |
| AI layer | None | GPT-4o, Quantum Fusion Engine |
| Consumer UX | Technical, intimidating | Consumer-friendly, mobile-responsive |
| Parlay tools | Basic | Full parlay builder + Life Changer |
| Admin suite | None | Fully built |

**Differentiation:** Unabated's user is a professional who already understands all the data. SORS MAXIMA targets the much larger segment of serious recreational bettors who want professional-grade insights without needing to understand EV notation.

### 2.3 OddsJam — Arbitrage Specialist

**Overview:** Focuses on arbitrage betting and positive EV betting. ~$49–$149/mo. Strong growth, reportedly $3M+ ARR as of 2023.

| Dimension | OddsJam | SORS MAXIMA |
|-----------|---------|-------------|
| Core use case | Finding arb and +EV opportunities | Pick intelligence, parlay building |
| Simulation engine | None | 51-factor Monte Carlo |
| Pick grades | EV-only | Full grade (S/A+/A/B+/B/C) |
| User profile | Arb-aware, arbitrage-seeking | Parlay-focused, analysis-seeking |
| Regulatory risk | Higher (arb can get accounts limited) | Lower (no arb, pure analytics) |
| Audience overlap | Moderate | Low — different buying trigger |

**Differentiation:** OddsJam solves a different problem (finding positive-EV lines) for a different user (arbitrage-aware bettors). SORS MAXIMA solves pick intelligence and parlay construction for the far larger recreational-to-serious segment.

---

### 2.4 BetQL — Low Cost, Low Quality

**Overview:** Consumer-facing pick service with data layer. Very affordable ($10–$30/mo). Large user base but low ARPU.

| Dimension | BetQL | SORS MAXIMA |
|-----------|-------|-------------|
| Price | $10–$30/mo | $49–$199/mo |
| Simulation engine | None | 51-factor Monte Carlo |
| Pick quality | Algorithm-based, no explanation | AI-grounded, fully transparent |
| ARPU | ~$18/mo | ~$89/mo |
| Target | Casual bettors | Serious bettors willing to pay |

**Differentiation:** BetQL competes on price. SORS MAXIMA competes on intelligence depth and transparency. At 5x the ARPU with a fraction of the users, SORS MAXIMA reaches the same revenue at lower scale.

---

### 2.5 Dimers — Free, Ad-Supported

**Overview:** Free sports betting tips site. Primarily editorial with some data. Ad-supported model.

**Differentiation:** Dimers is at the bottom of the market. Zero direct competition with a premium SaaS product like SORS MAXIMA. Users who use Dimers and want more will naturally upgrade to a paid product.

---

### 2.6 Sportradar / Stats Perform — B2B Data Giants

**Overview:** Enterprise sports data companies. Not consumer-facing. Clients are sportsbooks, media companies, broadcasters.

**Strategic Note:** These are **potential acquirers**, not competitors. Both companies have expressed interest in building consumer-facing analytics layers. Acquiring SORS MAXIMA would give either company an immediate consumer product to complement their B2B data stack.
